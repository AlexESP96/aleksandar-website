# Aleksandar Plavsic homepage

Static HTML/CSS homepage. Open `index.html` locally to preview.

## Deploy to Vercel
1. Create a GitHub account at github.com.
2. Create a new repository, e.g. `aleksandar-website`.
3. Upload these files to the repository.
4. Create a Vercel account at vercel.com.
5. Click Add New > Project.
6. Import the GitHub repository.
7. Keep the default settings and click Deploy.
8. Add your custom domain in Vercel Project Settings > Domains.
